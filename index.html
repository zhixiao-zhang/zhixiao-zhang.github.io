I am a forth-year undergraduate student at Sichuan University in Chengdu China, majoring in Software Engineering. I am interested in programming language, program analysis, compiler, computer system, and computer security. I am currently working on a static analysis to categorize pointer arithmetic operations, which is the basis of a dynamic analysis for the memory usage pattern.
My ultimate goal is to contribute to building efficient, secure and resilient system software from programming language perspective. While I may not directly participate in every project, I aim to develop tools and techniques that can support these goals across a wide range of applications.
